glaze

CSS-in-JS microlibrary for making design systems approachable with React

Get Started
Star

Familiar

Co-locating CSS with markup, glaze praises the component-based development era. Say goodbye to naming conventions with an API resembling inline styles.

Personalizable

Inspired by Theme UI, design tokens are first-class citizens. Enjoy frictionless autocompletion for various scales. Define convenient property aliases and shorthands on your own.

Lightweight

Built upon treat, styles are statically extracted with near-zero runtime by default. Dynamic style injection is an opt-in feature, providing enhanced flexibility at low cost.
